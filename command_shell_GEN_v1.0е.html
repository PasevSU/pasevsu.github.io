Shell Command Generator for Home Assistant

This tool helps you generate YAML files for the shell_command integration in Home Assistant

Important Usage Information

Configuration Format

This generator creates files for use with !include_dir_named:

# configuration.yaml
shell_command: !include_dir_named yaml/shell_command

Each file must contain only the command, without the shell_command: prefix!

Shell Command Instructions

  • Shell commands run in the Home Assistant environment
  • Use full paths to executable files
  • For complex commands, create a script and call it
  • Command name must not contain spaces
  • Files are saved in the yaml/shell_command/ directory

Command Settings

Name must not contain spaces!
Command is required!

Configuration Preview

File will be downloaded with .yaml extension
Configuration will appear here after clicking "Generate Configuration"
